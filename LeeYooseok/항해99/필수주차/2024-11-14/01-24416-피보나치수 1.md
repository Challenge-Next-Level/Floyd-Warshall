1. 문제 분석
- 주어진 수 n의 피보나치수를 두가지 방법을 통하여 구한다.
  - 재귀
    - fibo(n) = fibo(n - 1) + fibo(n - 2)
  - 동적 프로그래밍
    - dp[n] = dp[n - 1] + dp[n - 2]
  
- 각 방법의 실행 횟수를 구해라.


- 입력
  - 첫째 줄 n 

2. 제약조건
- n (5 <= n <= 40)

3. 사용 알고리즘
- 재귀, DP

4. 풀이 방법
- 재귀
  - 재귀 함수 구현 및 실행 횟수 확인
- DP
  - n - 2번
