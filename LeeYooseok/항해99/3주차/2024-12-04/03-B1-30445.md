1. 문제 분석
- 문자의 행복 점수 : (P_H / (P_H + P_G))
  - P_H : 문자에 {H, P, Y} 의 개수
  - P_G : 문자에 {S, D} 의 개수
  - P_H와 P_G 가 모두 0인 경우에 행복 지수는 0.5이다.


- 입력
  - 첫째 줄에 문자 메시지가 주어진다. 문자는 영문자 알파벳 단어로 구선된다. 단어는 공백으로 분리되어 주어진다. 

2. 제약조건
- 각 단어는 대문자로 구성된다. 단어의 최대 길이는 20이며, 최대 단어수는 80이다.

3. 사용 알고리즘
- 문자열

4. 풀이 방법
- 행복 및 슬픔 문자의 개수를 센다.

5. 어려운 점
- 숫자 나눗셈 (실수 처리 -> 부동 소수점 고려)
