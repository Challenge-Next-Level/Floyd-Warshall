- 입력
  - 첫째 줄에 스카이라인의 고도가 바뀌는 지점의 좌표 개수 n이 주어진다.
  - n개의 줄에 좌표 x와 y가 주어진다.

2. 제약조건
- 스카이라인의 고도가 바뀌는 지점의 좌표 개수 n : (1 <= n <= 50,000)
- 좌표 x, y : (1 <= x <= 1,000,000. 0 <= y <= 500,000)

4. 풀이 방법
