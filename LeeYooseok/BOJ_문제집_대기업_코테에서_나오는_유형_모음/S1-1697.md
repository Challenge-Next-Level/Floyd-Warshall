1. 문제 분석
- 수빈의 위치 N, 동생의 위치 K
- 수빈이는 걷거나 순간이동을 할 수 있다.
  - 걷기, X -> (X - 1 or X + 1)
  - 순간이동, X -> 2 * X
- 수빈이가 동생을 찾을 수 있는 가장 빠른 시간

- 입력
  - 첫째줄에 수빈의 위치 N과 동생의 위치 K가 사이에 공백을 두고 주어진다.

2. 제약조건
- 수빈의 위치 N : (0 <= N <= 100,000)
- 동생의 위치 K : (0 <= K <= 100,000)

3. 사용 알고리즘
- BFS

4. 풀이 방법
- 수빈이의 위치에서 시작해, 걷기, 순간이동을 탐색한다.

5. 어려운 점
- 처음에 DP로 풀어볼까 생각하다가, 그래프 탐색 알고리즘으로 풀어야 하는것을 알았다.
